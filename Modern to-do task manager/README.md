## Project Description

The Modern To-Do Task Manager is a web-based application developed using HTML, CSS, and JavaScript. It helps users organize and manage daily tasks efficiently through a simple and modern user interface.

## Features

* Add Tasks
* Edit Tasks
* Delete Tasks
* Mark Tasks as Completed
* Task Statistics
* Filter Tasks (All, Active, Completed)
* Local Storage Support

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Scope

This project demonstrates front-end web development concepts including DOM manipulation, event handling, and local storage management. It helps users maintain task lists and improve productivity.
